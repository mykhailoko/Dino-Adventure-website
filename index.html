ABOUT US

FEATURES:

We have 3 different dinosaurs and
6 skins for each of them!

You can play 4 different levels
with different obstacles!

Want to play with a friend? Let's go!
Everyone has 3 lives, the best one wins.

DEVELOPING:

The game was made in Python
using the Pygame library.

Our game is an open source project,
the code is in the GitHub repository.

Dino Adventure is a cross-platform game,
you can download it to any device.
